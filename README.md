# MYC

Get your YC application roasted in 10-minute sessions. No DMs. Just feedback.

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up Supabase:**
   - Go to [Supabase SQL Editor](https://supabase.com/dashboard/project/hoevksqthngrlsmrancx/sql)
   - Run the SQL from `supabase-schema.sql`

3. **Run development server:**
   ```bash
   npm run dev
   ```

4. **Deploy to Vercel:**
   ```bash
   vercel
   ```

## Environment Variables

Already configured in `.env.local`:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`

## Tech Stack

Next.js 16 • TypeScript • Tailwind CSS • Supabase
